# Tetris
A C++ Tetris Game utilizing SFML 2.6.2 and heavily adhering to Object Oriented Programming(OOP) principles.

---Welcome to 'Kuch Bikhre Huwe Blocks' (Tetris)---

The game consists of 7 unique shapes that fall from above and stack on top of each other.

The goal of the game is to prevent the blocks from reaching the top of the screen by completely filling
the rows which causes the row to collapse and brings the blocks down.

If a block reaches the top of the screen the game ends.

A score counter present at the left hand of the screen keeps track of the number of rows collapsed.

The upcoming shape is displayed on the right hand of the screen.

You can control the falling blocks by rotating and moving them so that they may fill a row.


Controls:
W - rotates the falling shape
A - moves the falling shape to the left
D - moves the falling shape to the right
S - moves the falling shape downward
SpaceBar - instantly moves the falling shape to the bottom most available space
R - Restarts the game once it has ended
